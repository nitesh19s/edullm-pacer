🧪 RAG Chat Testing Guide - Chemistry Grade 10

📋 Prerequisites:
  • ✅ NCERT Chemistry Grade 10 PDF uploaded
  • ✅ Data integrated (clicked "Integrate Data" button)
  • ✅ Ollama running with llama3.2 and nomic-embed-text models
  • ✅ Local models initialized in Settings

📖 Step-by-Step Testing Instructions

1️⃣ Open the Main Platform

Navigate to index.html in your browser

2️⃣ Initialize Local Models (if not done)
  • Go to Settings section (⚙️)
  • Scroll to "Local AI Models"
  • Click "Test Connection"
  • Verify: ✅ Ollama is running
  • Verify: ✅ Models available: llama3.2, nomic-embed-text
3️⃣ Go to RAG Chat Section
  • Click on "RAG Chat" in the navigation (💬)
  • You should see the chat interface with filters at the top
4️⃣ Set Filters for Chemistry Grade 10

In the chat header, set the following filters:

  • Subject Filter: Select Chemistry
  • Grade Filter: Select Grade 10
  • Source Filter: Select NCERT (or leave as "All Sources")
✅ Result: Now all RAG queries will ONLY search in Chemistry Grade 10 content!
5️⃣ Test with Chemistry Questions

Try these sample queries in the chat input:

Query 1: "What are chemical reactions?"
Expected: Should retrieve content about chemical reactions from Chemistry Grade 10
Query 2: "Explain the periodic table"
Expected: Should retrieve content about periodic table organization
Query 3: "What are acids and bases?"
Expected: Should retrieve content about acids, bases, and pH
Query 4: "Explain carbon compounds"
Expected: Should retrieve content about organic chemistry and carbon
Query 5: "What is corrosion?"
Expected: Should retrieve content about corrosion and rusting
6️⃣ Verify the Response

When you send a query, check:

  • ✅ Response appears after 3-10 seconds (local processing time)
  • ✅ Response contains relevant Chemistry content
  • ✅ Source citations show "Chemistry" and "Grade 10"
  • ✅ Console shows: "Filtered by subject 'chemistry': X results"
  • ✅ Console shows: "Filtered by grade '10': X results"
7️⃣ Check Browser Console

Open Developer Console (F12) and look for:

🔍 RAG Query Started: [your question]
✨ Query cleaned: [cleaned query]
📦 Retrieved X chunks
📚 Filtered by subject "chemistry": X results
🎓 Filtered by grade "10": X results
🤖 Using local model for RAG response
✅ RAG Query Complete (Xms)

🧪 Advanced Testing

Test Filter Switching
  1. Ask a Chemistry Grade 10 question with filters set
  2. Change Subject to "Physics" (keep Grade 10)
  3. Ask the same question again
  4. Verify: Response now comes from Physics instead of Chemistry!
Test Without Filters
  1. Set all filters to "All"
  2. Ask: "What is energy?"
  3. Verify: Results may come from multiple subjects (Physics, Chemistry, Science)

🔍 Troubleshooting

❌ Problem: "No relevant information found"

Solutions:
  • Check if Chemistry Grade 10 PDF was uploaded and integrated
  • Go to Upload History → verify Chemistry Grade 10 entry exists
  • Check console for indexing confirmation
  • Try re-integrating data (Data Upload → Integrate Data)
❌ Problem: Getting results from wrong subject/grade

Solutions:
  • Double-check filter dropdowns are set correctly
  • Refresh the page and re-apply filters
  • Check console logs for "Filtered by..." messages
❌ Problem: "Local models not initialized"

Solutions:
  • Go to Settings → Local AI Models
  • Click "Test Connection"
  • If Ollama not running: brew services start ollama
  • Refresh page after starting Ollama
